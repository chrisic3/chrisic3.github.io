Sample Blog

A simple blog template using HTML ans CSS.

Purfect Match Demo

A sample, fictional landing page integrating Bootstrap and Font Awesome. The page is responsive for mobile devices by utilizing Bootsptrap's collapsible navbar. All buttons and links go nowhere as this is is a sample in integrating Bootstrap.

Color Game

A color matching app created after learning Javascript and DOM manipulation. It presents the user with a random RGB color and they must select the correct color below. It also has an easy mode that limits the choices to three.

Todo App

A simple Todo app built with HTML, CSS, and jQuery. It allows the user to add new items, mark them as completed by adding a strikethrough, and delete items. It does not use a database to store the items, only appends a new li inside the ul.

Patatap Clone

A simplified clone of the patatap.com app using paper.js for the circles and howl.js for the sounds. Keys A-Z are each mapped to their own sound.

RESTful Blog

A simple blog app that allows the creation of new posts, editing of existing posts, and the deletion of posts. It was created while leaning the concepts of RESTful routing. Some of the technologies used in this app are Semantic UI, Node.js Express.js, and MongoDB. The app is hosted on Heroku using MLab for the production database.

YelpCamp

This is a Yelp type app in which users can post and view information about fictional campsites. It supports user registration, sign in and sign out, the creation, editing, and deleting of campsites, the creation, editing, and deletion of comments, and flash success and error messages. Users must sign in to create, edit, or delete campgrounds and comments, and only the user to which the campground or comment belongs to can edit or delete them. Some of the technologies used in this app are Bootstrap, Node.js, Express.js, Passport.js and MongoDB. The app is hosted on Heroku using MLab for the production database.
